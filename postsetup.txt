Last, perform the following steps manually:

# LibreWolf
- Run once: 'librewolf --setDefaultBrowser'
- In LibreWolf
  - browse to 'about:config', set 'privacy.resistFingerprinting' to 'false'
  - go to 'Settings'
    - select dark mode
    - under "Filter lists" tab, select the two entries under "Cookie notices"
  - install extension "SponsorBlock"

# Omarchy
- Change the Omarchy theme forth and back to "Tokyo Night" for window decorations to re-apply!
- Cycle through the wallpapers with SUPER CTRL + SPACE until the custom one appears
- Relaunch Omarchy for some changes to take effect
- Set up Fido (for sudo)
- Set up Web Apps

# Flutter
- Install Android Studio
  - In Omarchy: Install -> AUR -> android-studio (https://developer.android.com/studio/install#linux)
  - Install Android SDK via Android Studio -> SDK Manager -> SDK Tools -> select Android SDK Command-line Tools
- Install Flutter (https://docs.flutter.dev/get-started/install/linux)
- Run 
  - 'flutter --disable-analytics'
  - 'dart --disable-analytics'
  - 'flutter doctor'
- You may (!) need to run
  - 'flutter config --android-sdk "${HOME}/Android/Sdk"' (depending on installation path)
  - 'flutter config --android-studio-dir "${HOME}/Android/Studio"' (depending on installation path)
  - 'sudo usermod -a -G flutter $USER'
- Run 'flutter doctor --android-licenses'  

# Tailscale
- In Omarchy: Install -> Service -> Tailscale
